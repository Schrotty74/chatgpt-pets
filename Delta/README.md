# Delta

Ein freundliches, originales Sci-Fi-Delta-Abzeichen. Delta winkt, schwebt beim Verschieben im Warp-Flug, arbeitet konzentriert an seinem integrierten Kontrollpult und reagiert humorvoll auf Fehler.

## Installation

Den Ordner `Delta` herunterladen und nach `~/.codex/pets/delta` kopieren. Anschliessend Delta in Codex auswaehlen.
