# Delta – das verspielte Abzeichen

[English version](README.en.md)

Delta ist ein verspieltes Star-Trek-Delta-Abzeichen. Es winkt, fliegt mit kleinen Warp-Effekten, arbeitet konzentriert an einem Kontrollpult und reagiert mit humorvollen Bewegungen auf Fehler.
