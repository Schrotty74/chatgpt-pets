# Delta

<img align="left" width="220" src="preview-transparent.png" alt="Delta, a playful sci-fi delta badge">

An original friendly sci-fi delta badge. Delta waves, hovers in warp flight while moving, works at its integrated control panel, and reacts humorously to errors.

<br clear="left">

## Installation

Download the `Delta` folder, copy it to `~/.codex/pets/delta`, then select Delta in Codex.
