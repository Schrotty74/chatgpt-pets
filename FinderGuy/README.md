# Finder Guy

<img align="left" width="220" src="preview-transparent.png" alt="Finder Guy, ein freundliches blaues Maskottchen">

Finder Guy ist ein freundliches zweifarbiges blaues 3D-Maskottchen. Er reagiert mit klaren Blickrichtungen auf die Maus und begleitet Codex mit seinen vorhandenen Animationen.

<br clear="left">

## Installation

Den Ordner `FinderGuy` herunterladen und nach `~/.codex/pets/finderguy` kopieren. Anschliessend Finder Guy in Codex auswaehlen.
