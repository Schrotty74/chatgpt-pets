# Finder Guy

<img align="left" width="220" src="preview-transparent.png" alt="Finder Guy, a friendly blue mascot">

Finder Guy is a friendly two-tone blue 3D mascot. He follows the mouse with clear look directions and accompanies Codex with his existing animations.

<br clear="left">

## Installation

Download the `FinderGuy` folder, copy it to `~/.codex/pets/finderguy`, then select Finder Guy in Codex.
