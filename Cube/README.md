# Cube – der Zauberwürfel

[English version](README.en.md)

Cube ist ein freundlicher Zauberwürfel. Während er arbeitet, dreht er seine Ebenen und wird sichtbar Schritt für Schritt gelöst. Beim Warten denkt er über die nächste Drehung nach, bei Fehlern gerät er kurz durcheinander und am Ende präsentiert er stolz den vollständig gelösten Würfel.
