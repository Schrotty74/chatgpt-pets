# Woodi – der verspielte Codex-Otter

<img align="left" width="220" src="preview-transparent.png" alt="Woodi, ein Otter mit Holzstück">

[English version](README.en.md)

Woodi ist ein freundlicher, neugieriger Otter, der Codex mit einem kleinen Holzstück begleitet. Er winkt, bewegt sich, springt, wartet auf Eingaben, arbeitet konzentriert, prüft Ergebnisse und kann in alle Richtungen schauen.
