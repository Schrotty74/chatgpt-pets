# Inky – der verspielte Oktopus

<img align="left" width="220" src="preview-transparent.png" alt="Inky, ein verspielter blau-violetter Oktopus">

Inky spielt mit seinen acht Armen und wechselt seine Farben passend zum Zustand von Codex: beim Arbeiten wird er kräftig türkis-blau, beim Warten heller und bei Fehlern kurz gesprenkelt und verwirrt.

<br clear="left">

## Installation

Den Ordner `Inky` herunterladen und nach `~/.codex/pets/inky` kopieren. Anschliessend Inky in Codex auswaehlen.
