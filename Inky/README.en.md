# Inky – the playful octopus

<img align="left" width="220" src="preview-transparent.png" alt="Inky, a playful blue-purple octopus">

Inky plays with all eight arms and shifts color to match Codex’s state: rich turquoise-blue while working, lighter while waiting, and briefly mottled and confused after an error.

<br clear="left">

## Installation

Download the `Inky` folder, copy it to `~/.codex/pets/inky`, then select Inky in Codex.
