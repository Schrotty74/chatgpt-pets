# WonderCube – der denkende Zauberwürfel

<img align="left" width="220" src="preview-transparent.png" alt="WonderCube, ein freundlicher Zauberwürfel">

[English version](README.en.md)

WonderCube ist ein freundlicher Zauberwürfel, der sich während der Arbeit Schritt für Schritt sichtbar löst. Er denkt beim Warten über die nächste Drehung nach, gerät bei Fehlern kurz durcheinander und präsentiert am Ende stolz seine gelöste Seite.
