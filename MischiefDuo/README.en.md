# MischiefDuo – cat and mouse

<img align="left" width="220" src="preview-transparent.png" alt="MischiefDuo, a retro cat with mouse">

An original retro slapstick cat-and-mouse duo. They chase while moving, work together, wave, wait, and briefly land in a harmless comic mishap after an error.

<br clear="left">

## Installation

Download the `MischiefDuo` folder, copy it to `~/.codex/pets/mischiefduo`, then select MischiefDuo in Codex.
