# Byte – the construction robot

<img align="left" width="220" src="preview-transparent.png" alt="Byte, a small construction robot">

Byte carries bricks, checks plans, and carefully assembles a small tower while working. When something fails, the tower briefly topples before Byte starts building again.

<br clear="left">

## Installation

Download the `Byte` folder, copy it to `~/.codex/pets/byte`, then select Byte in Codex.
