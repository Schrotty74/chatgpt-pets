# Byte – der Bauroboter

[English version](README.en.md)

Byte ist ein kleiner Bauroboter. Er trägt Bausteine, prüft Baupläne und setzt während der Arbeit konzentriert ein kleines Bauwerk zusammen. Bei Fehlern fällt sein Turm kurz um, danach baut er weiter.
