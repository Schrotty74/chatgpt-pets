# Byte – der kleine Bauroboter

<img align="left" width="220" src="preview-transparent.png" alt="Byte, ein kleiner Bauroboter">

Byte trägt Bausteine, prüft seinen Bauplan und setzt während der Arbeit konzentriert einen kleinen Turm zusammen. Bei Fehlern kippt der Turm kurz um, bevor Byte weiterbaut.

<br clear="left">

## Installation

Den Ordner `Byte` herunterladen und nach `~/.codex/pets/byte` kopieren. Anschliessend Byte in Codex auswaehlen.
